   - Problem: Calculate simple interest given principal, rate, and time.

7. **Area of Circle**
   - Problem: Calculate the area of a circle given its radius.

8. **Maximum of Three Numbers**
   - Problem: Find the maximum of three given numbers.

9. **Leap Year Check**
   - Problem: Determine if a given year is a leap year.

10. **Reverse a String**
    - Problem: Reverse a given string.

11. **Count Vowels**

12. **Palindrome Check**
    - Problem: Check if a string is a palindrome.

13. **Linear Search**
    - Problem: Implement a linear search algorithm.

14. **Sum of Natural Numbers**
    - Problem: Calculate the sum of the first 'n' natural numbers.

15. **Multiplication Table**
    - Problem: Print the multiplication table of a given number.

16. **Fahrenheit to Celsius Conversion**
    - Problem: Convert temperature from Fahrenheit to Celsius.

17. **Calculate BMI**
    - Problem: Calculate the Body Mass Index (BMI) from height and weight.

18. **Greatest Common Divisor (GCD)**
    - Problem: Find the GCD of two numbers.

19. **Decimal to Binary Conversion**
    - Problem: Convert a decimal number to binary.

20. **Count Digits**

21. **Sum of Digits**

22. **Check Prime Number**

27. **Binary to Decimal Conversion**

28. **Largest Element in Array**

29. **Average of Numbers**

30. **Count Occurrences of a Character**

31. **Check if a String is Numeric**

32. **Concatenate Two Strings**

33. **Find the Second Largest Number in an Array**

## Medium Problems

34. **Binary Search**
   - Problem: Implement binary search algorithm.

35. **First Non-Repeated Character**
   - Problem: Find the first non-repeated character in a string.

36. **Bubble Sort**
   - Problem: Implement bubble sort algorithm.

37. **Insertion Sort**
   - Problem: Implement insertion sort algorithm.

38. **Merge Sort**
   - Problem: Implement merge sort algorithm.

39. **Find All Subsets**
   - Problem: Generate all subsets of a given set.

40. **Balanced Parentheses**
   - Problem: Check for balanced parentheses in an expression.

41. **Max Subarray Sum (Kadane’s Algorithm)**
   - Problem: Find the maximum sum of a contiguous subarray.

42. **Anagrams Check**
   - Problem: Determine if two strings are anagrams of each other.

43. **Permutations of a String**
    - Problem: Generate all permutations of a string.

44. **Longest Common Prefix**
    - Problem: Find the longest common prefix among a set of strings.

45. **Roman to Integer Conversion**
    - Problem: Convert a Roman numeral to an integer.

46. **Integer to Roman Conversion**
    - Problem: Convert an integer to a Roman numeral.

47. **Count Primes**
    - Problem: Count the number of prime numbers less than a non-negative number, n.

48. **Sieve of Eratosthenes**
    - Problem: Implement the Sieve of Eratosthenes to find all prime numbers up to a given number.

49. **Tower of Hanoi**
    - Problem: Solve the Tower of Hanoi puzzle.

50. **Next Permutation**
    - Problem: Implement next permutation, which rearranges numbers into the lexicographically next greater permutation of numbers.

51. **Power of Two**
    - Problem: Determine if a number is a power of two.

52. **Sqrt(x) Implementation**
    - Problem: Compute and return the square root of x.

53. **Pascal's Triangle**
    - Problem: Generate the first n rows of Pascal's triangle.

54. **Set Matrix Zeroes**
    - Problem: Given an m x n matrix, if an element is 0, set its entire row and column to 0.

55. **Search Insert Position**
    - Problem: Given a sorted array and a target value, return the index if the target is found. If not, return the index where it would be if it were inserted in order.

56. **Combination Sum**
    - Problem: Given a set of candidate numbers (candidates) and a target number (target), find all unique combinations in candidates where the candidate numbers sum to target.

57. **Rotate Image**
    - Problem: Rotate the image by 90 degrees (clockwise).

58. **Group Anagrams**
    - Problem: Given an array of strings, group anagrams together.

59. **Maximum Product Subarray**
    - Problem: Find the contiguous subarray within an array (containing at least one number) which has the largest product.

60. **Word Search**
    - Problem: Given a 2D board and a word, find if the word exists in the grid.

61. **Longest Consecutive Sequence**
    - Problem: Given an unsorted array of integers, find the length of the longest consecutive elements sequence.

62. **Valid Sudoku**
    - Problem: Determine if a 9x9 Sudoku board is valid.

63. **Bitwise AND of Numbers Range**
    - Problem: Given a range [m, n] where 0 <= m <= n <= 2147483647, return the bitwise AND of all numbers in this range, inclusive.

64. **Majority Element**
    - Problem: Find the majority element in an array which appears more than ⌊ n/2 ⌋ times.

65. **Task Scheduler**
    - Problem: Given a char array representing tasks CPU need to do. Each letter represents a different task. Tasks could be done without original order. Each task could be done in one interval. For each interval, CPU could finish one task or just be idle. However, there is a non-negative cooling interval n that means between two same tasks, there must be at least n intervals that CPU are doing different tasks or just be idle.

66. **Kth Largest Element in an Array**
    - Problem: Find the kth largest element in an unsorted array.

67. **Unique Paths in a Grid**
    - Problem: Count the number of unique paths from the top-left corner to the bottom-right corner of a m x n grid.
